TAKSwiftSupport
===========

Swift用汎用クラス集

##Installation with CocoaPods
###Podfile
```ruby
pod 'TAKSwiftSupport', '~> 0.1.2'
```

#### Use Exension
```ruby
pod 'TAKSwiftSupport/Exension', '~> 0.1.2'
```

###Requirements
requires iOS 8 & xCode7
