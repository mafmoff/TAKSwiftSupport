TAKSwiftSupport
===========

Swift用汎用クラス集

##Installation with CocoaPods

###Requirements
requires iOS 8 & xCode7

###Podfile
```ruby
pod 'TAKSwiftSupport', '~> 0.3.6'
```

#### Use Exension
```ruby
pod 'TAKSwiftSupport/Extension', '~> 0.3.6'
```

#### Use CoreMotion
```ruby
pod 'TAKSwiftSupport/CoreMotion', '~> 0.3.6'
```

### User Realm
```ruby
pod 'TAKSwiftSupport/Realm', '~> 0.3.6'
```
